## 🎯 Product Summary

Cognitive OS is an AI-powered Second Brain platform that enables users to capture, organize, search, and connect knowledge using semantic search, note tagging, and interactive knowledge graphs.

Target Users:
- Students
- Researchers
- Lifelong Learners

Core Features:
- AI Semantic Search
- Knowledge Graph
- Dashboard
- Note Organization
- JWT Authentication
- Tag-based Knowledge Management
